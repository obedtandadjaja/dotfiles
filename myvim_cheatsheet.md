# NERDTree
- Will automatically open when vim starts up with no files specified
- `<C-n>` for toggle

# Surround 
```
"Hello world" -> :cs"' -> 'Hello world'

'Hello world' -> :cs"( -> ( Hello world )

(Hello world ) -> :cs() -> (Hello world)

(Hello world) -> :ds) -> Hello world

Hello world -> cursor in Hello and :ysiw) -> (Hello) world

(Hello) world -> :yss) -> ((Hello) world)
```
