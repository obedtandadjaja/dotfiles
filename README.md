# Dotfiles
My dotfiles configuration.

## Configuration
* Prezto - https://github.com/sorin-ionescu/prezto
* neovim
* Alacritty
* fzf
* ag
* tmux

# Installation
Note: this command will replace all your settings with the setting in the repo!

```
chmod +x sync_to_local.sh
./sync_to_local.sh
```

# Backup
```
chmod +x sync_to_repo.sh
./sync_to_repo.sh
```
